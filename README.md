# Calculadora de factors de sostenibilitat
Aquesta eina és una calculadora docent pensada per facilitar l'anàlisi de formats digitals a partir de diversos factors de sostenibilitat relacionats amb la preservació digital. El seu objectiu és ajudar els estudiants a comprendre, aplicar i discutir conceptes com l'obertura de les especificacions, l'adopció, la transparència, l'autodescripció, les dependències externes, l'impacte de possibles restriccions de patents i la presència de mecanismes de protecció tècnica.

L'aplicació permet introduir una valoració per a cada factor, afegir observacions, justificacions i evidències, calcular una puntuació ponderada i generar alertes quan hi ha aspectes crítics, encara que el resultat global no sigui especialment baix. També incorpora visualitzacions i una fitxa d'exemple precarregada.

Els pesos relatius assignats als factors responen a un criteri expert i incorporen un component subjectiu. Per aquest motiu, els resultats s'han d'entendre com un suport per a l'anàlisi i la discussió, no com una forma d'avaluació universal o definitiva dels formats.

## Requisits
- PHP 8.0 o superior
- Servidor web (Apache, Nginx o similar)

## Posada en marxa
1. Descomprimeix el projecte.
2. Situa'l dins del teu servidor local.
3. Obre `index.php` al navegador.
